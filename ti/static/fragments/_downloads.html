Installation and Startup

  1. Click the "Download Evaluation Demo" button above and select a location to save the resulting .zip file.
  2. Uncompress the .zip file that was just downloaded. In most operating systems, this is accomplished by simply double-clicking the .zip file.
  3. Navigate inside the extracted folder named "mct-demo-1.8" and double-click the file named "MCT.jar" Alternately, from the command line you can navigate within the extracted folder and run this command: java -jar MCT.jar
  4. The Open MCT evaluation demo should now start and run.

Installation and Process

Deploying Open MCT in your environment is a process which will vary depending on your specific needs, but can be summarized by a few straightforward steps as noted below:

  1. Install MCT on your workstation or a shared drive.
  2. Create a shared MySQL database for MCT using the included creation scripts.
  3. Install or author plugins to add the features you need, such as connections to data sources.
  4. Modify configuration files to locate your servers and fine-tune MCT's features to your needs.

To get started, and for more information, see the MCT Developer Wiki at https://github.com/nasa/mct/wiki
