Start experiencing Open MCT today

Download and experiment with the evaluation demo that includes preloaded objects, an embedded database, a selection of views, and a simulated data feed. If you haven't viewed our overview video yet, we encourage you to do so - it's the best way to get familiar with MCT's capabilities and features.

Evaluation Version

System Requirements

  • Windows 7 or greater; Mac OS 10.8.5 or later; most Linux systems.
  • Java SE 6 or later.
  • Some demo features require an Internet connection.

Download Evaluation Demo

Installation and Startup

  1. Click the "Download Evaluation Demo" button above and select a location to save the resulting .zip file.
  2. Uncompress the .zip file that was just downloaded. In most operating systems, this is accomplished by simply double-clicking the .zip file.
  3. Navigate inside the extracted folder named "mct-demo-1.8" and double-click the file named "MCT.jar" Alternately, from the command line you can navigate within the extracted folder and run this command: java -jar MCT.jar
  4. The Open MCT evaluation demo should now start and run.
