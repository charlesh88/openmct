Mission Control Technologies – Open Sourced

Control centers for space missions are tasked with operating spacecraft. While the specific tasks may vary with each mission, the basic roles - monitoring spacecraft health and status, sending commands to carry out mission goals, monitoring activities, knowing where the spacecraft is relative to its planetary or deep space objective - are similar across missions. Traditionally, software for conducting space mission operations is built using traditional application architectures, using a suite of different applications. Open MCT changes that paradigm completely.

What is Open MCT?

Developed to support situational awareness for space missions, Open MCT eliminates the walls associated with traditional software by replacing applications with compositions. Users are provided with an integrated environment of pre-built software objects from which software may be assembled by putting the building blocks together. It’s a simple as drag and drop.
